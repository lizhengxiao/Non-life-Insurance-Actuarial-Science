10.2 索赔

在非寿险精算中，索赔和损失是不同的。 索赔是指保险事故发生后，由投保人根据保险合同的规定向保险人提出的赔付请求。由于保险合同中通常存在免赔额、赔偿限额、共保等保险条款，索赔通常是低于损失的。注意，在很多精算模型中，索赔和损失有时被用于表示完全相同的含义。

索赔通常包括索赔次数、索赔金额和累积赔款。如果考虑风险单位的影响，分别对应索赔频率、索赔强度和纯保费。

  • 索赔次数表示被保险人在一定时期内发生索赔的次数。索赔频率在一定时期内（通常为一年）的索赔次数，通常用总索赔次数和风险单位数之比进行估计：

\[ \text{索赔频率}=\frac{\text{总索赔次数}}{\text{风险单位数}} \]

10.3 Example

  • 一个汽车保单组合在 2014 年有 5000 个车年的风险单位数，该年发生的索赔次数为 800 次， 那么在该年平均每个车年的索赔频率估计值为 800/5000 = 16%

  • 一份汽车保险单在 5 年内发生了 6 次索赔，那么这份保单的索赔频率估计值为 6/5 = 1.2 次，即每年发生 1.2 次索赔

10.4 赔款(loss, claim)

  • 赔款或损失：保险人已付或应付给索赔人的金额，有时也被称作赔付额（claim amount, claim size）

  • 赔款 = 已付赔款（paid loss）+ 未决赔款

    • 未决赔款：

      • 个案准备金 (case reserve) or 应付赔款（payable loss) - 已发生已报案的预期成本

      • 已发生未完全报案赔款 (incurred but not enough reported, IBENR) - 对个案准备金可能不足的调整

      • 已发生未报案赔款 (incurred but not reported, IBNR)

  • 应付赔款（payable loss)：保险人已经接到报案而预期在未来将要支付的赔款，通常表现为个案准备金（case reserve）

10.5 赔款(loss, claim)

  • 已报案赔款：某一事故年所有已付赔款和个案准备金之和。

    • 已报案赔款 = 已付赔款 + 个案准备金

  • 最终赔款（ultimate loss）= 已报案赔款 + IBNR 准备金 + IBENR 准备金

10.7 理赔费用（loss adjustment expenses）

  • 理赔费用：可以分解为直接理赔费用和间接理赔费用。

    • 直接理赔费用(allocated loss adjustment expenses, ALAE)：与特定索赔直接相联系的费用。如特定索赔的堪察费用和辩护费用等。

    • 间接理赔费用(unallocated loss adjustment expenses, ULAE)：与特定索赔不直接相联系的费用。如理赔部门管理人员的薪资、办公费用、数据处理费用

  • 在厘定保险费率时，通常将与放在一起作为一个整体来处理。(按照一定比例分配)

10.9 保费（premium）

  • 保费 = 赔款（纯保费） + 费用附加 + 利润附加

  • 费用是指保险公司支出的承保费用、管理费用和理赔费用等，通常区分为固定费用和变动费用两类。

    • 固定费用是指与纯保费无关的费用

    • 变动费用是指与纯保费的变动直接相关的费用

  • 利润附加反映了保险公司经营保险业务应该获取的利润水平，传统上通常将其表示为保费的一定百分比。

10.10 赔付率（loss ratio）

  • 赔付率：赔款除以保费。

    • 赔款可以是已付的、已报案的或预测的最终赔款；

    • 保费可以是承保保费或已赚保费。

  • 损失率与赔付率的区别：

    • 在计算赔付率时，分子必须是赔款，分母必须是保费。

    • 在计算损失率时，不必如此严格。譬如，如果在一年的保险期间内，如果有20％的风险单位遭受了损失，可以认为损失率就是20％。
